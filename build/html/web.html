How to make a small web¶

  1. Login in Chat360 or create a account - https://app.chat360.io/login

  2. Click on ADD NEW CHATBOT > START FROM SCRATCH > Website

  1. Below Image Shows Components and Pages

  1. You can use different components to create chatbot according to your requirements and Save

  1. If you don’t have website, create your website on https://www.blogger.com/ for this example

  2. Remember your website name

  3. Now go to chat360 > chatbot canvas > Publish > Enter Website Name and Publish

  1. Now you will get Script > Copy that Script

  1. And paste this script in your website html theme

  1. Paste the Script At Bottom Above </body> and Save

  1. Now go to website > you will see the chatbot on your website published
